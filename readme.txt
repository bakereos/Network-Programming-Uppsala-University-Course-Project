Create a network game using winsock with features like, server finding, interpolating positions, input prediction, etc. The game must have a win/lose condition and should be playable by two people on different devices on the same network. 


 To start playing locally, run both the client and server projects by configuring startup projects on the solution, and selecting both of them, then running.

 To play the game press C to connect to the server. Use WASD to move around. Hold Spacebar to boost. Boosting is the input and action the player can do. Hold Z to see connection information. Press X to disconnect 

The game I made is one where bombs spawn randomly on the screen and you try not to hit them. Hitting them a few times lowers your hp to 0 and you die. Survive the 20 seconds and you win. If playing with another player, the player with the most hp by 20 seconds wins.
